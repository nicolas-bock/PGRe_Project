# PGRe_Project

## Installation
### Steps
1. **Clone the Repository**
   ```bash
   git clone https://github.com/nicolas-bock/PGRe_Project
   cd PGRe_Project
   ```

2. **Create ``build`` file**
   ```bash
   mkdir build
   cd build
   ```

3. **Run CMake to generate a buildsystem**
   ```bash
   cmake ..
   ```

5. **Build and run the project using the make build system**
   ```bash
   make -j8 && make run
   ```
